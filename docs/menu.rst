.. _menu:

ShotGrid Menu in Alias
######################################

The :class:`tk_alias.menu_generation.AliasMenuGenerator` class is responsible for creating the ShotGrid menu and adding it to the Alias menubar. It manages adding and removing the ShotGrid menu items to launch the Engine's configured Toolkit Apps inside of Alias.
