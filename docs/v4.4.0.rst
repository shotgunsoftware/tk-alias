v4.4.0
=====================

Release highlights
------------------

* Alias 2027.0 support
* Python 3.11 support
* Fix overriding Alias construction options when importing a file
* Fix setting context when using "Open Source File" operation in Alias

Dependencies
---------------

* Alias framework: ``tk-framework-alias v2.6.0``
    * Required for Alias 2027.0 support
    * Required for Python 3.11 support
    * Required for fixing overriding Alias construction options when importing a file

Bug Fixes
-----------------

* Fixed issue with overriding Alias construction options when importing a file:
    * Previously when importing an Alias file (e.g. using the Loader App or the API `import_file` function), the construction options from the imported file were overriding the current file construction options. This is now fixed, and the construction options from the imported file will not override the current file construction options.
* Fixed issue with setting context when using "Open Source File" operation in Alias:
    * Previously when using the "Open Source File" operation in Alias, the Task information was not retrieved for the context. This caused issues when trying to save or publish the file after opening from "Open Source File" operation. This is now fixed, and the Task information will be retrieved for the context.
